# PUMA 560 Kinematics Simulator

## Overview

This project implements the kinematic modeling and visualization of the PUMA 560 robotic manipulator using Denavit-Hartenberg (DH) parameters.

The objective is to demonstrate manipulator kinematics, coordinate frame assignments, transformation matrices, and robotic arm motion through computational simulation and visualization.

Based on concepts from Chapter 3 of John J. Craig's *Introduction to Robotics: Mechanics and Control*.

## Features

* Denavit-Hartenberg Parameter Modeling
* Forward Kinematics Computation
* Transformation Matrix Generation
* Coordinate Frame Visualization
* Multi-Link Robot Modeling
* Interactive Robot Manipulator Simulation
* PUMA 560 Robot Representation
